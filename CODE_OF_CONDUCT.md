# 社区行为准则

Shadoc 社区致力于为每位参与者提供友善、安全、尊重且不受骚扰的协作环境，不因经验、身份、背景或观点而区别对待。

## 期望行为

- 围绕事实、技术和用户影响展开讨论；
- 尊重不同经验与观点，给出具体且可执行的反馈；
- 主动保护日志、凭据、内部路径和其他个人或敏感信息；
- 在发现错误时善意提醒，并愿意修正自己的行为；
- 把安全漏洞通过私密渠道报告，而不是公开披露可利用细节。

## 不可接受的行为

- 骚扰、威胁、歧视、侮辱或人身攻击；
- 未经许可公开他人的身份、联系方式、凭据或私密资料；
- 故意破坏项目、绕过安全边界，或鼓励危险的数据操作；
- 在被明确要求停止后继续进行令人不适的互动；
- 其他在专业协作环境中明显不当的行为。

## 执行

普通协作问题可通过 GitHub Issue 提醒维护者；涉及安全、隐私或需要保密处理的事件，请使用 GitHub Private Vulnerability Reporting。维护者可以编辑或删除不当内容，限制参与权限，并在必要时向平台报告。

维护者会尽量私密、公平地调查报告，并只向处理事件所必需的人员披露信息。善意报告者不会因提交问题而受到报复。
